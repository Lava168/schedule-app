# 日程安排工具 - 界面设计文档

## 设计理念

本应用遵循 Apple Human Interface Guidelines (HIG)，打造一款简洁、直观的日程管理工具。采用移动端竖屏 (9:16) 布局，支持单手操作，让用户能够快速创建、查看和管理日程事件。

## 屏幕列表

### 1. 日历主页 (Calendar Home)
- **主要内容**: 月视图日历网格，显示当前月份
- **功能**: 
  - 切换月份（左右滑动或点击箭头）
  - 点击日期查看当日事件
  - 有事件的日期显示标记点
  - 底部显示选中日期的事件列表

### 2. 日程列表 (Schedule List)
- **主要内容**: 按日期分组的事件列表
- **功能**:
  - 显示今日及未来事件
  - 快速预览事件时间、标题
  - 点击进入事件详情

### 3. 事件详情 (Event Detail)
- **主要内容**: 单个事件的完整信息
- **功能**:
  - 显示标题、日期、时间、描述
  - 显示提醒设置
  - 编辑和删除按钮

### 4. 创建/编辑事件 (Create/Edit Event)
- **主要内容**: 事件表单
- **功能**:
  - 输入标题（必填）
  - 选择日期
  - 选择开始/结束时间
  - 添加描述（可选）
  - 设置提醒时间
  - 保存/取消按钮

## 主要用户流程

### 创建新事件
1. 用户在日历主页点击右上角 "+" 按钮
2. 进入创建事件页面
3. 填写事件标题、选择日期和时间
4. 可选：添加描述、设置提醒
5. 点击保存，返回日历主页
6. 日历上对应日期显示事件标记

### 查看和编辑事件
1. 用户在日历主页点击某个日期
2. 底部显示该日期的事件列表
3. 点击某个事件进入详情页
4. 点击编辑按钮进入编辑模式
5. 修改后保存

### 删除事件
1. 在事件详情页点击删除按钮
2. 弹出确认对话框
3. 确认后删除事件并返回

## 组件设计

### 日历网格
- 7列（周日到周六）
- 6行（覆盖完整月份）
- 今日日期高亮显示
- 选中日期使用 primary 颜色圆形背景
- 有事件的日期下方显示小圆点

### 事件卡片
- 圆角矩形，surface 背景
- 左侧显示时间
- 右侧显示标题和简短描述
- 有提醒的事件显示铃铛图标

### 表单输入
- 标准 iOS 风格输入框
- 日期/时间选择器使用原生组件
- 清晰的标签和占位符文字

## 数据模型

```typescript
interface Event {
  id: string;
  title: string;
  date: string; // YYYY-MM-DD
  startTime: string; // HH:mm
  endTime: string; // HH:mm
  description?: string;
  reminder?: number; // 提前提醒分钟数 (0, 5, 10, 15, 30, 60)
  createdAt: string;
  updatedAt: string;
}
```

## 本地存储

使用 AsyncStorage 存储事件数据，无需后端服务器。数据结构：
- `@schedule_events`: 事件数组 JSON

## 提醒功能

使用 expo-notifications 实现本地提醒：
- 创建事件时，根据提醒设置安排本地通知
- 编辑事件时，更新或取消已有通知
- 删除事件时，取消相关通知
